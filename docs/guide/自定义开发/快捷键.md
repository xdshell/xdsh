# 快捷键

Xdsh可以注册自定义的按键。目前已有的快捷键有'Enter'，'Tab’，'Ctrl+l'。

## 语法

```javascript
this.xdsh.registerHotkey(name, func, ctrl=false, alt=false);
```

## 栗子

注册'Ctrl+l'

```javascript
this.registerHotkey('l', (event)=>{
  event.preventDefault();

  this.cmdline.history.clear();
}, true);
```
